## 🚀 Getting Started

1. **Install OpenClaw:** Follow [official docs](https://docs.openclaw.ai)
2. **Clone this repo:**
   ```bash
   git clone https://github.com/eNudimmud/enki-openclaw.git ~/.openclaw/workspace
   ```
3. **Customize:**
   - Copy `USER.md.example` to `USER.md` and fill in your details
   - Update `TOOLS.md` with your local environment specifics
   - Adjust `SOUL.md` and `IDENTITY.md` to match your persona
4. **Run:** `openclaw gateway start`
